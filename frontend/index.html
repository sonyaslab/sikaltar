Input Harga
Harga berlaku dan harga konstan 2010 per komoditas per wilayah. Harga konstan hanya dapat diisi pada tahun dasar (2010).
→ Buka S1.H
 🌾
S1.P
Input Produksi
Kuantum produksi per komoditas. Mendukung input per triwulan dan tahunan langsung dengan deteksi konflik otomatis.
→ Buka S1.P
 📈
S1.R
Rasio SUT
Rasio OS, WIP, KA, ADJ. Default dari SUT 2019 BPS. Override lokal dengan preview estimasi dampak NTB sebelum simpan.
→ Buka S1.R
 📉
S1.I
Indeks Deflator
Indeks harga implisit (basis 2010=100) untuk kategori yang menggunakan metode Deflasi. Perubahan % dari tahun lalu otomatis.
→ Buka S1.I
